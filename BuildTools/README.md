# BuildTools - Button UI Dependencies

This directory contains the Button tool and its dependencies for creating console-based UI elements.

## Files

- **Button.bat** - Main Button function for creating GUI buttons
- **Box.bat** - Function for drawing boxes
- **Getlen.bat** - Function for getting string length
- **batbox.exe** - Console manipulation utility
- **GetInput.exe** - Mouse and keyboard input handler
- **quickedit.exe** - QuickEdit mode controller

## Source

All files are from the [Button project](https://github.com/Batch-Man/Button) by Kvc/Batch-Man.

## Usage

These tools are used by `BuildUI.bat` to provide a configuration interface for the Windows Setup Helper build process.
